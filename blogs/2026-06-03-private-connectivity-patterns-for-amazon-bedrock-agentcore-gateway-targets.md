This post covers four private connectivity patterns for AgentCore Gateway including MCP server targets via private endpoints, REST API targets using VPC Lattice, Regional API Gateway targets through VPC Link, and Lambda targets using Hyperplane ENIs. Hybrid and multi-cloud connectivity options are also addressed.
